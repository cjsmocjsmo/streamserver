# Motion Detection Stream Server - Dependencies
#
# This project uses APT packages only to comply with PEP 668 
# (externally-managed-environment) on modern Debian/Ubuntu systems.
#
# DO NOT use pip install on these packages - use apt instead:
#
# Install all dependencies:
#   sudo apt update
#   sudo apt install python3-opencv python3-numpy python3-picamera2
#
# Or run the setup script:
#   python3 setup.py
#
# Package mappings (pip name -> apt package):
# opencv-python -> python3-opencv
# numpy -> python3-numpy  
# picamera2 -> python3-picamera2
#
# Note: All required packages are available in Debian/Ubuntu repositories.
# No pip installation or virtual environments are needed for this project.
